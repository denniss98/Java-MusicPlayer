Steps to setup the directories (NetBeans tutorial)

1. Go to project side tab
2. Go to source packages
3. Right click
4. Properties
5. Run
6. Arguments
7. Enter music folder directory e.g C:\Users\USERNAME\Desktop\MusicPLayer\music 
8. Enter sample file directory next to it e.g. C:\Users\USERNAME\Desktop\MusicPLayer\sample_list.txt

Example
C:\Users\USERNAME\Desktop\MusicPLayer\music C:\Users\USERNAME\Desktop\MusicPLayer\sample_list.txt

With one space inbetween the two directories
